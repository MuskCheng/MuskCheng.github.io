# 写在前面

我们上网都在看些什么？

随着21世纪互联网的爆发式增长，海量的信息每天都在不断地轰击着我们的大脑。举个例子，如今我们每天所接触到的信息，已经远超古人一生所接触到的信息的总和。

这些乱七八糟的信息对我们真的有用么？

只不过是在别人聊八卦的时候，让自己不掉队的筹码罢了。

# 信息分类

互联网每天都在生成数以亿计的信息：新闻、视频、博客文章、有声读物、播客、小说以及各类社交媒体的短文、短视频等等。

> [!NOTE]
>
> 互联网所作的似乎就是把我们的专注和思考能力撕成碎片，抛到一边。
>
> 上网时间越多，阅读中篇文章就越难集中注意力。有些人担心自己正在患上慢性注意力分散症。

当我们所有的阅读都集中在屏幕上进行的时候，我们的阅读量就变少了，是的，这很反常识，读的越多，读的越少，但事实就是这样。

随着互联网应用数量的不断增长，人们开始逐渐的把消遣的工具转向了电子屏幕，看视频，聊八卦，发感慨，以及各种莫须有的内容。

> [!IMPORTANT]
>
> 随着**AI**技术的不断发展，甚至不敢想象**Deepseek**此类跨越式进步的**AI**技术，也仅仅发布了不到一年的时间，**AI**生文、生图甚至生成视频的能力已经几乎达到以假乱真的地步，你在互联网上看到的大部分内容可能就是有人使用**AI**无中生有的。

转头想想，你已经有多长时间没有认真的阅读过一篇文章了，哪怕是一篇完整的新闻报道？

纸媒的没落是一个时代的落幕，但真正有价值的信息依旧藏在各种长文本和书籍之中。

# 信息密度

中文是信息密度极高的一种语言，但是，信息密度再高，几句话又能表达的了多少内容呢？甚至连新闻里最基本的时间、地点、人物都表达不全，更不用说讲讲清楚事情的前因后果了。

短视频、微博一类的社交媒体，其上所承载各种内容不仅全无信息量，甚至会把你引导到完全错误的方向上去。

举个例子，总有一些人会使用AI将网络上那些原本复杂的教程简化抽象为一个小片段，以此来拉高自己账号的点击量，但实际上，这些教程根本就无法使用，你看多了甚至会产生自我怀疑，为什么我就做不到？

这当然不是你做不到，而是那些教程缺斤短两，AI怎么会知道教学的重点在哪里？他们把重点都省略了，挑拣出来的片段又怎么会达到最终效果呢？甚至他们自己都没有尝试过，你在评论区的提问会被一些莫名其妙的言论所埋没，这些东西根本就不值得去学习，甚至多看一眼都是在浪费时间。

# 信息传递与阅读习惯

互联网的发明极大的缩短了人们获取信息的速度，大洋彼岸的信息也可以在一瞬间弹出在你的眼前，无惧时间与距离因素，极大的拉近了我们与世界的距离，这场变革对于人类来说是史无前例的，人类上一次信息爆发式增长还是在**活字印刷术**发明之后，人人都可以建立自己的小型图书馆。

> [!NOTE]
>
> 事实上，自从图书馆创立以来，公共图书馆一直都是最重要、最有影响力的信息传媒之一，只有在默读习惯和活字印刷术出现之后，这一传媒才开始蓬勃发展。

但是对比起图书馆，互联网仿佛一个庞然大物，目前人类最大的图书馆的馆藏图书也仅仅是互联网的一隅罢了，而各地图书馆，研究所的文献馆也开始接入互联网提供服务，现代图书馆你遇到更多可能是敲键盘，看屏幕，而翻页声也将离我们越来越远。

尽管电子书已经被大张旗鼓的鼓吹了很久了，但是纸质书依然是相当一部分人的热爱，比如我自己，我就是一个忠实的纸质书爱好者，但是我也看电子书，尤其是亚马逊推出了**Kindle**这种划时代的电子屏幕之后，那种类纸质的屏幕质感吸引了很大一部分纸质书爱好者，我也是其中之一。但是我的阅读习惯是先查找电子书下载观看，感觉是优质的图书，我会买上一本纸质书进行收藏，只有纸质书才能提醒我自己，曾经我看过的、且认为有用的一切。

尽管手机与**Kindle**有着重叠甚至超越的阅读功能，但是**Kindle**有一样东西是手机永远**无法企及**的，那就是**专注阅读**的能力，是的，手机、平板亦或是电脑这类电子屏幕根本无法做到这一点，他们拥有太多能够容易吸引人去别处的功能了，绝对的自律是不存在的，我想这点各位应该都明白。

# 思维方式

有研究表明，人类的大脑会不断的被各种生活方式所改造，你用一种方式学习、娱乐的时候，你的大脑在不断强化这种形式的过程中，大脑的思维链也在不断的试错、重组，形成新的、更适合这种方式的思维链。

> [!TIP]
>
> 大脑是很容易被塑造的，但是前提的你需要不断地强化这个过程，大脑不会因为年龄的增长就停滞不前。

高密度的信息传递自然需要高密度的信息载体作为支撑，当我们的大脑被各种低密度信息高密度轰炸后，我们的大脑自然也会发生一些变化。

有没有发现自己现在已经很难静下心来去看完一篇文章甚至超过30分钟的长视频**（娱乐节目除外）**了？

# 互联网是如何困住我们的

能够吸引人们驻足的并不只有优美的事物，还有感官刺激。那些想要吸引人驻足的视频，在标题、开头，留足了各种爆炸性的信息，可能与实际脱节，甚至完全不契合实际，他们的目的只有一个，就是要让你留下，只有你留下的时间足够长，他们的流量就会足够大，只有流量足够大，广告的效益才能达到最大化。只要能够有收益，他们不会在乎给你提供的内容是否有用。

我们每天都被包裹在各式各样的信息里，各个互联网企业还会不停的更新自己的推荐算法，我们也需要明白一个道理，算法是比我们更懂我们自己的，有时候，我们是不是怀疑过他们利用我们手机在监听我们的一切，但实际上，只是我们的自己的行为出卖了自己罢了。

> [!tip]
>
> 只要特征足够细致，哪怕没有任何私密性信息，算法也会推断出我们是谁，算法会给这些特征添加标签，你和这些特征的契合程度越高，你是这个人的可能性也就越大，可别忘了，你在所有互联网企业的软件里都是进行过实名认证的，美其名曰保卫网络安全，但电信诈骗是怎么精准的匹配到你的个人信息的？甚至比公安户籍信息知道的都要全面？

算法深深的了解我们的各种喜好，然后给我们进行强化推荐，你的每一次驻留，都给算法增加了给你打上标签的概率值，然后我们就被算法困住了。

> [!CAUTION]
>
> 要知道，从现在开始，你所能够看到的，就不再是全部了，只是他们认为你“想”看到的“一切”罢了。
>
> 这就是**信息茧房**:lock:。
