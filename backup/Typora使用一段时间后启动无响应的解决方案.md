1. Win+R打开运行窗口，输入regedit，回车，启动注册表编辑器
2. 按以下路径

```
计算机\HKEY_CURRENT_USER\Software\Typora
```

* 点击右键，选择权限

![](https://knowledgeimg.eddycheng.online/img/2026/01/20260129140649920.png)

* 在组或用户名处找到并选中Administrator，完全控制处勾选拒绝即可

![](https://knowledgeimg.eddycheng.online/img/2026/01/20260129141124937.png)
